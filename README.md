# animeGanv2_for_M2Pro
Based on AnimeGan2 video to animation video plug-in library, support mac M2 pro accelerated operation.

### **Requirements**

#### python 3.9

#### tensorflow-macos
#### tensorflow-metal
#### opencv
#### tqdm
#### numpy
#### glob
#### argparse
#### onnxruntime (If onnx file needs to be run.)
